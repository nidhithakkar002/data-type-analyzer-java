# Data Type Analyzer (Java)

A console-based Java project that analyzes user input and demonstrates:

## 🔍 Features
- Detects data type (int, long, double)
- Performs type conversions
- Demonstrates overflow behavior
- Shows binary representation

## 🧠 Concepts Used
- Primitive data types
- Type casting
- Wrapper classes
- Autoboxing & Unboxing
- Overflow handling
- Binary representation

## 🚀 Example
Input:
130

Output:
- Detected Type: int
- byte: -126 (overflow occurred)
- short: 130
- long: 130
- Binary representation shown
